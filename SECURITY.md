# Security Policy

## Supported Versions

We actively support and patch security vulnerabilities for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

---

## Reporting a Vulnerability

If you discover a potential security vulnerability in Glacier.Chrono, please **do not** open a public issue. Instead, report it privately through one of the following methods:

1. **GitHub Security Advisories**: Submit a private draft advisory via the repository's "Security" tab.
2. **Email**: Send details of the vulnerability to `security@glacier-chrono.example.com` (use PGP if necessary to encrypt your report).

Please include the following information in your report:
* A detailed description of the vulnerability.
* Steps to reproduce the issue (including proof-of-concept code if available).
* The potential impact of the issue.

We will acknowledge receipt of your report within 48 hours and work with you to coordinate a security fix and public disclosure timeline.
